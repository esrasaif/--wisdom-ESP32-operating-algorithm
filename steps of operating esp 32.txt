
steps to operating esp32
1-install arduino ide from the website 
2- import the library of esp32 acourding this steps : go to file>Preferences>at "additional boards manager urls" paste this link to import the library https://dl.espressif.com/dl/package_esp32_index.json       
      3- Now we have to define the board acourding this steps : tools>Board Arduino Uno >boards manager >in the search icon write "ESP 32"  > install >  close
4-select the port : tools   > port  > select the port like COM6
5-now we can operating any code
